# 线程与并发占位说明

## 1. 这一章准备讲什么

Java 线程这一章，不是只背几个方法名，而是先分清：

- 什么是进程，什么是线程
- 为什么会有并发问题
- 线程是怎么创建和运行的
- 线程之间怎么通信
- 线程安全问题到底是怎么来的
- 面试里最常问哪些并发基础点

也就是说，这一章最容易混的不是单个 API，而是：

- 线程基础概念
- 线程生命周期
- 线程安全和可见性
- 锁和并发工具的整体关系

## 2. 为什么这一章适合放在这里

因为前面已经补到了：

- 集合
- 数据结构
- 异常
- IO
- 泛型

这些内容更偏“单线程基础”。

而真正到了面试阶段，很多问题会开始往下追：

- 线程怎么创建
- `sleep` 和 `wait` 区别
- `synchronized` 和 `Lock` 区别
- `volatile` 能不能保证线程安全
- 线程池为什么比手动创建线程更常用

所以现在开线程这一章，是比较自然也比较有面试价值的。
