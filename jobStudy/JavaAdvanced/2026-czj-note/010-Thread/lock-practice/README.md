## 1. 这一份怎么用

1. 先看锁专题知识文档

## 3. 当前阶段建议先练哪几类

当前阶段最值得先练的通常有：

1. 两线程交替打印
2. 三线程顺序打印
3. 生产者消费者
4. 固定顺序执行
5. 用 `synchronized` 和 `ReentrantLock` 各写一版

## 4. 做题时重点看什么

做锁题时，不要只盯着“代码能不能跑”。

更应该看：

- 谁在等谁
- 谁负责唤醒谁
- 锁对象是不是一致
- 条件判断是不是放在循环里
- 有没有丢通知风险
- 有没有死锁风险
