# Redis 牛客公开面经题

## 1. 这个子目录是干什么的

这个子目录专门整理牛客公开面经里反复出现的 Redis 面试题。

它不重复讲 Redis 基础原理，而是重点解决：

- 牛客上 Redis 最常被问哪些题
- 哪些是基础题，哪些是项目题
- 面试时怎么把这些题讲得更顺

## 2. 当前目录结构

- `redis-nowcoder-high-frequency.md`
- `redis-nowcoder-answer-pack.md`
- `redis-nowcoder-mock-questions.md`
- `redis-nowcoder-last-10-minutes.md`

## 3. 怎么看最顺

建议顺序：

1. `redis-nowcoder-high-frequency.md`
2. `redis-nowcoder-answer-pack.md`
3. `redis-nowcoder-mock-questions.md`
4. `redis-nowcoder-last-10-minutes.md`

## 4. 一句话记住

> 牛客上的 Redis 题目并不散，主要反复集中在“数据类型、持久化、缓存问题、一致性、为什么快、淘汰策略、分布式锁、主从哨兵集群、项目场景”这几条线。
