**LynDrive** 是一款专为企业级能力设计的文件管理系统，提供目录管理、分块文件上传以及基于角色的访问控制（RBAC）。系统采用现代化的前后端分离架构：后端基于 Spring Boot，前端使用 Vue 3，并遵循 **“前端结构优先”** 的策略，在稳定用户交互的同时，逐步集成真实的后端服务。

## 系统能力

系统围绕几个核心业务线构建，负责处理文件和用户权限的完整生命周期：

- **文件与目录工作台**：支持目录树浏览、文件列表展示以及标准文件操作。
- **上传管道**：提供分块上传、断点续传，并基于 MD5 实现秒传（文件去重）。
- **RBAC 与安全**：管理用户、角色和权限，并提供专门的治理控制台。
- **目录治理**：实现资源级访问控制及权限继承审计。
- **审计日志**：追踪系统活动，并提供操作日志的过滤视图。

## 核心技术栈

| 层级       | 技术                                                          |
| :--------- | :------------------------------------------------------------ |
| **后端**   | Java 17, Spring Boot 3.3.5, MyBatis-Plus, Redis, MySQL, Hutool |
| **前端**   | Vue 3, Vite, Element Plus, Vue Router 4, Spark-MD5           |
| **存储**   | 本地文件系统（`data/storage`、`data/chunk-temp`）              |
