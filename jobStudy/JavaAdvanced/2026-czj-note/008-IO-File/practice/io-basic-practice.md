## 1. 这一组题主要练什么

这一组练习不是为了把 IO 写得特别复杂，而是先把最核心的主线练顺：

1. `File` 和流的区别
2. 字节流和字符流的区别
3. 缓冲流和转换流的作用
4. 对象流和打印流的基本理解
5. `try-with-resources` 的使用感觉
