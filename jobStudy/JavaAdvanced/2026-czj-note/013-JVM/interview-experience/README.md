# interview-experience

## 1. 这个子目录是干什么的

这个子目录专门放 JVM 的“外部面经整理版”。

和前面的主线笔记、mock 训练层不一样，这里更关注：

- 最近公开面经里 JVM 到底问什么
- 哪些题是反复出现的
- 这些题一般会怎么继续追问

## 2. 当前有哪些文档

- `jvm-interview-experience-high-frequency.md`
- `jvm-interview-experience-answer-notes.md`

## 3. 建议怎么使用

建议顺序：

1. 先看 `jvm-interview-experience-high-frequency.md`
   先知道公开面经里 JVM 高频题大概集中在哪几条线。
2. 再看 `jvm-interview-experience-answer-notes.md`
   练“这类题在真实面试里怎么答、怎么接追问”。

## 4. 一句话收尾

> 这个子目录不是再补教材，而是把公开面经里的真实问法重新整理成可复习、可作答的版本。
