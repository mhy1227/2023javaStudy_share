
### 题 10 下面代码里，为什么说这是泛型方法

```java
public <T> void printValue(T value) {

}
```

### 题 11 下面代码里，`Generator<String>` 表示什么

```java
interface Generator<T> {
    T next();
}

class StringGenerator implements Generator<String> {
    public String next() {
        return "hello";
    }
}
```
