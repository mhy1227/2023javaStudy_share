# 其余数据结构练习复习清单

## 1. 栈与队列是否过关

至少要会：

1. 栈的后进先出
2. 队列的先进先出
3. 双端队列
4. 用两个栈实现队列
5. 用队列实现栈
6. 有效括号
7. 最小栈
8. 单调栈 / 单调队列的基本感觉

## 2. 树是否过关

至少要会：

1. 前序遍历
2. 中序遍历
3. 后序遍历
4. 层序遍历
5. 最大深度
6. 翻转二叉树
7. 对称二叉树
8. 平衡二叉树
9. 最近公共祖先
10. 判断 BST
11. 能按模板手写 DFS / BFS / BST 校验 / LCA
12. 能口述树和 BST / 堆 / Hash / TreeMap 的区别

## 3. 堆是否过关

至少要会：

1. 大顶堆和小顶堆
2. PriorityQueue 默认行为
3. 第 K 大元素
4. 最小的 K 个数
5. 数据流中的中位数
6. TopK 题型

## 4. 如果面试只剩一轮复习时间

优先顺序：

1. 有效括号
2. 最小栈
3. 层序遍历
4. 判断 BST
5. 最近公共祖先
6. 第 K 大元素
7. 数据流中的中位数

## 5. 一句话总结

practice-2 这一组的重点，是把栈队列、树、堆这三类结构的核心题型先建立起来，后面再往更复杂题扩展。
