# 数据结构复习清单

## 1. 第一轮先复习什么

建议先按这条线复习：

1. 数据结构总览
2. 数组
3. 链表
4. 栈与队列
5. 树
6. 树的遍历
7. 二叉搜索树
8. 树进阶补充
9. 堆
10. 数据结构横向对比

## 2. 每种结构都要问自己的问题

复习每个结构时，都尽量回答下面这些问题：

1. 它是什么
2. 它是线性结构还是非线性结构
3. 它最擅长什么操作
4. 它最不擅长什么操作
5. 它和 Java 哪些集合有关

## 3. 数组要会什么

至少要能说清：

1. 数组元素类型相同
2. 数组长度固定
3. 数组适合按下标访问
4. 数组中间插入删除往往要移动元素
5. `ArrayList` 和数组思想强相关

## 4. 链表要会什么

至少要能说清：

1. 链表由节点连接形成
2. 链表不强调连续存储
3. 链表更关注连接关系
4. 链表不适合按下标直接访问
5. `LinkedList` 和链表思想强相关

## 5. 栈和队列要会什么

至少要能说清：

1. 栈是后进先出
2. 队列是先进先出
3. 双端队列两头都能操作
4. 栈和队列强调的是进出规则
5. `Deque`、`Queue`、`ArrayDeque` 都要有印象

## 6. 树要会什么

至少要能说清：

1. 树是非线性结构
2. 树适合表示层级关系
3. 根节点、父节点、子节点、叶子节点这些术语
4. 二叉树是最常见入门树结构
5. `TreeSet`、`TreeMap` 和树思想有关

## 7. 树的遍历要会什么

至少要能说清：

1. 前序是根左右
2. 中序是左根右
3. 后序是左右根
4. 层序遍历是一层一层访问
5. 遍历经常和递归、队列一起出现

## 8. 堆要会什么

至少要能说清：

1. 数据结构堆不是 JVM 堆内存
2. 堆常和完全二叉树联系在一起
3. 堆强调堆顶优先级
4. 分为大顶堆和小顶堆
5. `PriorityQueue` 和堆思想强相关

## 9. 二叉搜索树要会什么

至少要能说清：

1. 二叉搜索树是带有序规则的二叉树
2. 左边通常比根小，右边通常比根大
3. 中序遍历结果通常有序
4. 它和有序集合思想强相关
5. 它可能退化，因此后面还会学平衡树

## 10. 当前阶段最容易混的点

这几个最容易混：

1. 数据结构里的堆 和 JVM 堆内存
2. 栈这种数据结构 和 方法调用栈
3. 树 和 堆
4. 数组 和 `ArrayList`
5. 链表 和 `LinkedList`
6. 树结构有序 和 Hash 结构快速定位

## 11. 如果面试只让你快速讲一遍

你可以用下面这个顺序：

1. 先讲线性结构和非线性结构
2. 再讲数组、链表、栈、队列
3. 然后讲树、堆、二叉搜索树
4. 最后把它们和 Java 集合对应起来

## 12. 一句话总结

这一章复习的重点，不是把定义背得很复杂，而是知道每种结构“长什么样、擅长什么、和 Java 哪些集合相关”。
