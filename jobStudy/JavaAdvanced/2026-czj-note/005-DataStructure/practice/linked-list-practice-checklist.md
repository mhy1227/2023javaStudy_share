# 链表练习复习清单

## 1. 链表基础操作是否过关

至少要会：

1. 遍历链表
2. 头部插入
3. 尾部插入
4. 删除头结点
5. 删除尾节点
6. 删除指定值节点
7. 查找某个值
8. 统计链表长度

## 2. 虚拟头结点是否过关

至少要会判断这些题适不适合用虚拟头结点：

1. 删除指定值节点
2. 删除所有等于某值的节点
3. 删除倒数第 n 个节点
4. 删除重复元素

## 3. 快慢指针是否过关

至少要会：

1. 找中间节点
2. 找倒数第 k 个节点
3. 判断是否有环
4. 找环入口
5. 判断回文链表

## 4. 反转链表是否过关

至少要会：

1. 整体反转链表
2. 指定区间反转
3. k 个一组翻转

## 5. 合并类题是否过关

至少要会：

1. 合并两个有序链表
2. 合并 k 个有序链表
3. 相交链表

## 6. 删除类题是否过关

至少要会：

1. 删除指定值
2. 删除所有指定值
3. 删除倒数第 n 个节点
4. 删除排序链表重复元素 I
5. 删除排序链表重复元素 II

## 7. 综合题是否有印象

至少要见过：

1. 判断回文链表
2. 链表相加
3. 复制随机指针链表
4. LRU 缓存结构
5. 单链表排序
6. 链表重排

## 8. 如果面试只剩一轮复习时间

优先顺序：

1. 反转链表
2. 合并两个有序链表
3. 快慢指针
4. 环和环入口
5. 删除倒数第 n 个节点
6. 回文链表

## 9. 当前阶段最容易混的点

1. 当前节点和前驱节点
2. 删除节点和删除节点前驱
3. 找中点和找倒数第 k 个
4. 环检测和环入口
5. 普通删除和虚拟头结点删除

## 10. 一句话总结

链表题复习到位的标志，不是题做得多，而是你看到题目以后能很快判断：该不该用虚拟头结点、要不要快慢指针、是不是要做链表反转。
