# 链表操作练习题

## 1. 这一组练习主要练什么

这一组不是为了上来就做很难的链表综合题，而是为了先把链表最基础的操作练熟。

重点放在：

- 节点定义
- 指针移动
- 头结点处理
- 插入节点
- 删除节点
- 遍历打印
- 边界判断

如果这一层不熟，后面学：

- 反转链表
- 快慢指针
- 环检测
- 合并链表

会很容易乱。

## 2. 做题前先记住一个前提

链表和数组最大的区别在于：

- 数组更关注下标
- 链表更关注节点连接关系

所以链表题通常不是在想：

- 第几个位置怎么移动元素

而是在想：

- 当前节点是谁
- 下一个节点是谁
- 要不要断开原连接
- 要不要重新接上

## 3. 基础操作题

### 题 1 遍历单链表

给定一个单链表：

```text
1 -> 2 -> 3 -> 4 -> null
```

要求：

- 依次打印每个节点的值

思考点：

1. 遍历从哪里开始
2. 循环终止条件是什么

### 题 2 头部插入节点

给定链表：

```text
1 -> 2 -> 3 -> null
```

要求：

- 在头部插入值为 `0` 的新节点
- 结果变成 `0 -> 1 -> 2 -> 3 -> null`

思考点：

1. 新节点的 `next` 应该先指向谁
2. 头结点最后应该更新成谁

### 题 3 尾部插入节点

给定链表：

```text
1 -> 2 -> 3 -> null
```

要求：

- 在尾部插入值为 `4` 的节点

思考点：

1. 如何找到尾节点
2. 空链表时应该怎么处理

### 题 4 在指定节点后插入

给定链表：

```text
1 -> 2 -> 4 -> null
```

要求：

- 在值为 `2` 的节点后面插入 `3`
- 结果变成 `1 -> 2 -> 3 -> 4 -> null`

思考点：

1. 新节点应该先连向谁
2. 原节点再连向谁

### 题 5 删除头结点

给定链表：

```text
1 -> 2 -> 3 -> null
```

要求：

- 删除头结点
- 结果变成 `2 -> 3 -> null`

思考点：

1. 头结点如何更新
2. 空链表和单节点链表怎么处理

### 题 6 删除尾节点

给定链表：

```text
1 -> 2 -> 3 -> null
```

要求：

- 删除最后一个节点
- 结果变成 `1 -> 2 -> null`

思考点：

1. 为什么通常需要找到尾节点前一个节点
2. 单节点链表时返回什么

### 题 7 删除指定值的第一个节点

给定链表：

```text
1 -> 2 -> 3 -> 2 -> 4 -> null
```

要求：

- 删除值为 `2` 的第一个节点
- 结果变成 `1 -> 3 -> 2 -> 4 -> null`

思考点：

1. 删除的是头结点时怎么办
2. 删除中间节点时前驱节点如何处理

### 题 8 查找某个值是否存在

给定链表：

```text
3 -> 7 -> 9 -> 2 -> null
```

要求：

- 判断值 `9` 是否存在
- 判断值 `10` 是否存在

思考点：

1. 最直接的做法是什么
2. 时间复杂度大概是多少

### 题 9 找链表长度

给定链表：

```text
5 -> 6 -> 7 -> 8 -> null
```

要求：

- 返回链表长度

思考点：

1. 计数变量什么时候加一
2. 空链表长度是多少

### 题 10 反转打印链表

给定链表：

```text
1 -> 2 -> 3 -> 4 -> null
```

要求：

- 按 `4 3 2 1` 的顺序输出

思考点：

1. 是真的反转链表，还是只是反向输出
2. 可以借助栈吗

## 4. 进阶一点的小题

### 题 11 删除链表中所有等于某个值的节点

给定链表：

```text
1 -> 2 -> 2 -> 3 -> 2 -> 4 -> null
```

要求：

- 删除所有值为 `2` 的节点
- 结果变成 `1 -> 3 -> 4 -> null`

思考点：

1. 连续多个目标值怎么处理
2. 头部连续删除时怎么写更稳

### 题 12 找中间节点

给定链表：

```text
1 -> 2 -> 3 -> 4 -> 5 -> null
```

要求：

- 找到中间节点

思考点：

1. 快慢指针怎么走
2. 为什么快指针走两步、慢指针走一步

### 题 13 删除倒数第 n 个节点

给定链表：

```text
1 -> 2 -> 3 -> 4 -> 5 -> null
```

要求：

- 删除倒数第 `2` 个节点
- 结果变成 `1 -> 2 -> 3 -> 5 -> null`

思考点：

1. 如何把“倒数”转成“正向遍历可处理”
2. 为什么经常要用虚拟头结点

## 5. 做这一组题最容易犯的错

1. 忘了处理空链表
2. 忘了处理单节点链表
3. 删除头结点时写乱
4. 修改指针顺序出错，导致链断掉
5. 遍历条件写错导致空指针
6. 把“当前节点”和“前驱节点”混在一起

## 6. 一句话总结

链表基础操作题的核心，不是背代码模板，而是始终看清楚：当前节点是谁、前驱是谁、下一个是谁、连接关系该怎么改。
