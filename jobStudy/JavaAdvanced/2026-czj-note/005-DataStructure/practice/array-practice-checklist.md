# 数组练习复习清单

## 1. 数组基础操作是否过关

至少要会：

1. 删除第一个元素
2. 删除最后一个元素
3. 删除指定位置元素
4. 删除指定值
5. 头部插入
6. 指定位置插入
7. 数组反转
8. 查找最大值和最小值

## 2. 原地操作题是否过关

至少要会：

1. 删除有序数组中的重复项
2. 移除指定元素
3. 移动零
4. 旋转数组
5. 合并两个有序数组

## 3. 双指针是否过关

至少要会：

1. 两数之和
2. 三数之和
3. 盛水最多的容器
4. 接雨水
5. 有序数组平方后重新排序

## 4. 滑动窗口是否过关

至少要会：

1. 长度最小的子数组
2. 滑动窗口最大值
3. 最长无重复子数组

## 5. 前缀和是否过关

至少要会：

1. 区间和查询
2. 和为 K 的子数组
3. 寻找数组中心下标

## 6. 二分查找是否过关

至少要会：

1. 标准二分查找
2. 搜索插入位置
3. 找第一个和最后一个位置
4. 搜索旋转排序数组
5. 旋转数组中的最小值
6. 寻找峰值

## 7. 二维数组是否过关

至少要会：

1. 二维数组中的查找
2. 螺旋矩阵
3. 顺时针旋转矩阵
4. 矩阵置零

## 8. 综合题是否有印象

至少要见过：

1. 最小的 K 个数
2. 数组中的第 K 大元素
3. 合并区间
4. 缺失的第一个正整数
5. 数组中的逆序对
6. 买卖股票的最好时机
7. 最大子数组和

## 9. 如果面试只剩一轮复习时间

优先顺序：

1. 原地操作
2. 双指针
3. 滑动窗口
4. 二分查找
5. 前缀和
6. 二维数组

## 10. 一句话总结

数组题复习到位的标志，不是做过很多题，而是你看到题目以后能很快判断：这是原地操作、双指针、滑动窗口、前缀和、二分，还是矩阵模拟。
