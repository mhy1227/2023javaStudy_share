# 引用类型基础笔记

## 1. 什么是引用类型

除了基本数据类型之外，Java 中大量数据都属于引用类型。

常见引用类型有：

- 类
- 接口
- 数组
- `String`

例如：

```java
String s = "hello";
int[] arr = {1, 2, 3};
```

## 2. 引用类型和基本类型的核心区别

### 基本类型

- 变量里保存的是值

### 引用类型

- 变量里保存的是对象引用

这是最重要的区别。

## 3. 例子

### 基本类型

```java
int a = 10;
int b = a;
b = 20;
System.out.println(a); // 10
```

### 引用类型

```java
StringBuilder sb1 = new StringBuilder("abc");
StringBuilder sb2 = sb1;
sb2.append("def");
System.out.println(sb1); // abcdef
```

这里说明：

- `sb1` 和 `sb2` 指向同一个对象
- 改的是同一个对象内部内容

## 4. `null` 和引用类型

引用类型可以赋值为：

- `null`

例如：

```java
String s = null;
```

这表示：

- 当前没有指向任何对象

## 5. 为什么理解引用类型很重要

因为后面这些知识都和它直接相关：

- 对象
- 参数传递
- `final` 修饰引用
- `String` 常量池
- 集合中的对象操作

## 6. 当前阶段最该掌握的结论

1. 引用类型变量保存的是引用，不是对象本体
2. 多个引用可以指向同一个对象
3. `null` 只适用于引用类型
4. `String` 也是引用类型

## 7. 一句话总结

理解引用类型，关键是分清“变量”和“对象”不是一回事。
