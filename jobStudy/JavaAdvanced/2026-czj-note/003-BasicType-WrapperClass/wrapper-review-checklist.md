# 基本类型与包装类复习检查清单

## 1. 基本数据类型

你应该能直接说出 Java 的 8 个基本数据类型：

- `byte`
- `short`
- `int`
- `long`
- `float`
- `double`
- `char`
- `boolean`

自测标准：

- 不看资料也能完整写出
- 知道默认整数是 `int`
- 知道默认小数是 `double`

## 2. 基本类型和引用类型的区别

你应该能说清：

- 基本类型保存的是值
- 引用类型保存的是引用
- `String`、数组、对象都属于引用类型

## 3. 包装类映射关系

你应该能熟练对应：

- `int` -> `Integer`
- `double` -> `Double`
- `char` -> `Character`
- `boolean` -> `Boolean`

重点：

- `char` 对应 `Character`
- 不是 `Char`

## 4. 为什么需要包装类

你应该能回答：

- 为什么 Java 不直接只保留基本类型

标准理解：

- 很多场景只能使用对象
- 集合和泛型不能直接放基本类型
- 所以需要包装类来桥接“值”和“对象”

## 5. 自动装箱与拆箱

你应该会判断下面代码分别发生了什么：

```java
Integer a = 10;
int b = a;
```

自测：

- `Integer a = 10;` 是自动装箱
- `int b = a;` 是自动拆箱

## 6. 转换方法

你应该能分清：

- `parseXxx()`
- `valueOf()`
- `xxxValue()`

当前阶段必须记牢：

- `parseInt("123")` -> `int`
- `Integer.valueOf("123")` -> `Integer`
- `num.intValue()` -> `int`

## 7. 默认值

你应该知道下面的区别：

- 基本类型成员变量有默认值
- 包装类成员变量默认值是 `null`
- 局部变量都必须先赋值再使用

例如：

```java
int a;       // 成员变量默认 0
Integer b;   // 成员变量默认 null
```

## 8. 强制类型转换

你应该会判断：

- 哪些场景需要强转
- 强转会不会丢精度
- 会不会溢出

最常见的理解：

- 大转小通常要强转
- 强转可能截断小数
- 强转也可能发生溢出

## 9. 业务字段类型选择

你应该能回答：

- 为什么手机号通常用 `String`
- 为什么身份证号、订单号、学号也常用 `String`

核心原因：

- 它们更像标识，不是拿来做数学运算的值

## 10. final 与引用

你应该能说清：

- `final` 修饰基本类型变量，值不能再变
- `final` 修饰引用类型变量，引用不能再指向别处
- `final` 不等于对象不可变

## 11. String 在这一章中的定位

你应该知道：

- `String` 不是基本类型
- `String` 是引用类型
- `String` 是对象
- `String` 是不可变对象

## 12. Integer 的特殊点

你应该至少知道：

- `Integer` 存在缓存机制
- 常见缓存范围是 `[-128,127]`
- 所以 `==` 在某些值范围里可能出现“看起来相等”的情况

## 13. 当前阶段达标标准

这章复习达标，不要求你背所有细节，但至少要做到：

1. 基本类型和包装类关系说得清
2. 自动装箱拆箱会判断
3. `valueOf / parseXxx / xxxValue` 不混
4. `final` 和对象不可变的区别说得清
5. 业务字段为什么用 `String` 说得清

## 14. 一句话总结

如果你能把“类型分类 + 包装类作用 + 转换方法 + 强转 + String 定位”说顺，这一章就算过关了。
