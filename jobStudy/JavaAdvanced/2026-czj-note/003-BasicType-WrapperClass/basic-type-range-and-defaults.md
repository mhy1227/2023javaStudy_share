# 基本数据类型的范围、字节数与默认值

## 1. 为什么这一块还值得补

这部分在面试里不一定每次都深问，但非常常见的问法有：

- Java 有几种基本数据类型
- `int`、`long`、`double` 各占多少字节
- 默认值是什么
- `long` 和 `float` 为什么常写 `L`、`F`

所以这份文档的目标不是让你死背极限数值，而是让你先把高频基础点记牢。

## 2. 八种基本数据类型

Java 一共有 8 种基本数据类型：

- `byte`
- `short`
- `int`
- `long`
- `float`
- `double`
- `char`
- `boolean`

## 3. 字节数与常见范围

### 3.1 `byte`

- 1 字节
- 常见范围：`-128 ~ 127`

### 3.2 `short`

- 2 字节
- 范围比 `byte` 大

### 3.3 `int`

- 4 字节
- 整数默认最常用类型

### 3.4 `long`

- 8 字节
- 比 `int` 范围更大

### 3.5 `float`

- 4 字节
- 单精度浮点数

### 3.6 `double`

- 8 字节
- 双精度浮点数
- 小数字面量默认类型

### 3.7 `char`

- 2 字节
- 用来表示单个字符

### 3.8 `boolean`

- 逻辑类型
- 只有 `true` 和 `false`

注意：

- `boolean` 的存储细节不需要现在死抠
- 当前阶段重点记住它不是数值类型

## 4. 成员变量默认值

当它们作为成员变量时，常见默认值如下：

- `byte` -> `0`
- `short` -> `0`
- `int` -> `0`
- `long` -> `0L`
- `float` -> `0.0F`
- `double` -> `0.0`
- `char` -> `'\u0000'`
- `boolean` -> `false`

这里最容易记错的是：

- `char` 默认不是空格，也不是 `'0'`

## 5. 局部变量没有默认值

这个非常重要。

例如：

```java
public void test() {
    int a;
    // System.out.println(a); // 报错
}
```

说明：

- 成员变量有默认值
- 局部变量必须先赋值后使用

## 6. 为什么 `long` 常写 `L`

例如：

```java
long num = 1234567890123L;
```

原因是：

- 整数字面量默认按 `int` 处理
- 数值较大时，需要显式告诉编译器这是 `long`

所以平时建议：

- `long` 字面量后面加 `L`

## 7. 为什么 `float` 常写 `F`

例如：

```java
float score = 12.5F;
```

原因是：

- 小数字面量默认是 `double`

所以如果变量是 `float`，通常要写：

- `F`

## 8. 复习时最该记住的高频点

当前阶段建议你先记这些，而不是强背所有极值：

1. `byte` 1 字节，范围 `-128 ~ 127`
2. `short` 2 字节
3. `int` 4 字节
4. `long` 8 字节
5. `float` 4 字节
6. `double` 8 字节
7. `char` 2 字节
8. `boolean` 只有 `true/false`
9. `long` 字面量常写 `L`
10. `float` 字面量常写 `F`

## 9. 面试里常见的问法

### 问：Java 有哪些基本数据类型

答：

- 一共有 8 个，分别是整数、浮点、字符、布尔这四类

### 问：默认整数和默认小数是什么类型

答：

- 默认整数是 `int`
- 默认小数是 `double`

### 问：局部变量有没有默认值

答：

- 没有
- 必须先赋值后使用

## 10. 一句话总结

这部分最重要的不是死背所有边界值，而是把“8 种类型、默认值、字节数、`L/F` 后缀、成员变量和局部变量区别”记稳。
