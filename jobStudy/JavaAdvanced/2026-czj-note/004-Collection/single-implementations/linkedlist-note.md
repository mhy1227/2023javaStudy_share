# LinkedList 单体笔记

## 1. LinkedList 是什么

`LinkedList` 是 `List` 的一个常见实现类，同时它也常被用作：

- `Deque`
- 队列

所以它不只是一个普通列表。

## 2. 核心特点

- 属于 `List`
- 有序
- 可重复
- 还能承担队列和双端队列语义

## 3. 为什么它比较特殊

因为它经常会在两个场景里出现：

1. 作为 `List`
2. 作为 `Queue / Deque`

这也是它和 `ArrayList` 很不一样的地方。

## 4. 什么时候会想到它

适合这些场景：

- 既想当列表用
- 又有队列或双端操作需求

## 5. 常见方法

除了 `List` 常用方法，还常见：

- `addFirst()`
- `addLast()`
- `pollFirst()`
- `pollLast()`

## 6. 当前阶段注意点

- 它不是日常最默认的列表主力
- 但在带有队列语义的题里经常出现

## 7. 一句话总结

`LinkedList` 是一个兼具列表和双端队列特点的集合实现类。
