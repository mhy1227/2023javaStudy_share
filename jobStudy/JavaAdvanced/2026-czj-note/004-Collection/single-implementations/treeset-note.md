# TreeSet 单体笔记

## 1. TreeSet 是什么

`TreeSet` 是 `Set` 的一个实现类，它的突出特点不是普通去重，而是：

- 排序

## 2. 核心特点

- 不可重复
- 更偏排序
- 和比较规则强相关

## 3. 为什么它和普通 Set 不一样

因为它不是单纯存进去就完了，而是会更强调：

- 元素之间如何比较

所以它经常和：

- 自然排序
- 定制排序

联系在一起。

## 4. 什么时候会想到它

适合这些场景：

- 想去重
- 又想让元素自动保持某种顺序

## 5. 当前阶段注意点

- `TreeSet` 的重点不是“快不快”，而是“排序规则”
- 后面学比较器和排序时，这个类会再次出现

## 6. 一句话总结

`TreeSet` 是“带排序能力的去重集合”。
