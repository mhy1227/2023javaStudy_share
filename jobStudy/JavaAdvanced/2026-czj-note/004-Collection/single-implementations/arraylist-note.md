# ArrayList 单体笔记

## 1. ArrayList 是什么

`ArrayList` 是最常见的 `List` 实现类。

它的定位可以先简单记成：

- 有序
- 可重复
- 日常开发里最常用的列表容器

## 2. 核心特点

- 底层基于数组思想
- 支持按索引访问
- 允许重复元素
- 允许存放 `null`

## 3. 什么时候优先想到它

适合这些场景：

- 存一批有顺序的数据
- 允许重复
- 需要频繁遍历
- 需要按下标取值

## 4. 常用方法

- `add()`
- `get()`
- `set()`
- `remove()`
- `size()`

## 5. 当前阶段注意点

- 它是 `List` 主力实现类
- 集合题里如果没有特殊说明，默认第一反应通常就是 `ArrayList`

## 6. 一句话总结

`ArrayList` 是“普通列表场景”的默认主力集合。
