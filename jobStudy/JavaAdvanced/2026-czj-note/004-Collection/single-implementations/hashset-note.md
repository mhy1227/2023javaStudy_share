# HashSet 单体笔记

## 1. HashSet 是什么

`HashSet` 是最常见的 `Set` 实现类。

它的核心定位是：

- 去重

## 2. 核心特点

- 不可重复
- 无序
- 底层和哈希思想相关
- 允许存放 `null`

## 3. 什么叫无序

这里的无序主要是指：

- 迭代输出顺序通常不等于插入顺序

## 4. 什么时候优先想到它

适合这些场景：

- 普通去重
- 判断某元素是否已出现
- 不关心遍历顺序

## 5. 为什么它重要

因为它会把前面对象章节里的知识真正用起来：

- `equals()`
- `hashCode()`

## 6. 当前阶段注意点

- `HashSet` 是 `Set` 主力实现类
- 去重题里第一反应通常就是它

## 7. 一句话总结

`HashSet` 是“不关心顺序时的去重主力”。
