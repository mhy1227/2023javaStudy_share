# TreeMap 单体笔记

## 1. TreeMap 是什么

`TreeMap` 是 `Map` 的一个实现类，它最大的特点是：

- 更偏按 key 的规则排序

## 2. 核心特点

- 存 `key-value`
- key 不可重复
- 更偏排序
- 和 key 的比较规则强相关

## 3. 为什么它和 HashMap 不一样

因为它的重点不只是“能存映射”，而是：

- key 会按某种规则组织

所以它常和：

- 自然排序
- 比较器

联系在一起。

## 4. 什么时候会想到它

适合这些场景：

- 想通过 key 建映射
- 又希望 key 保持排序状态

## 5. 当前阶段注意点

- `TreeMap` 的核心关键词不是“哈希”，而是“排序”

## 6. 一句话总结

`TreeMap` 是“带排序能力的 Map”。
