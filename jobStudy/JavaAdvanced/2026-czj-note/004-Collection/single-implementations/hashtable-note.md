# Hashtable 单体笔记

## 1. Hashtable 是什么

`Hashtable` 是一个比较早期的 `Map` 实现类。

现在提到它，更多是为了：

- 认识集合历史
- 做面试对比

## 2. 核心特点

- 存 `key-value`
- 线程安全
- 比较老
- 对 `null` 更严格

## 3. 为什么现在还要知道它

因为面试里经常会问：

- `HashMap` 和 `Hashtable` 区别是什么

所以它虽然不是日常主力，但不能完全陌生。

## 4. 当前阶段定位

你现在先记住：

- 它是早期线程安全 `Map`
- 现在通常不是主流首选

## 5. 一句话总结

`Hashtable` 是“早期线程安全 Map”的代表类。
