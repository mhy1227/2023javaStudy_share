# List 基础笔记

## 1. 什么是 List

`List` 是 `Collection` 下面最常见的一类集合。

它的典型特点是：

- 有序
- 可重复

## 2. 什么叫有序

有序的意思不是“自动排好序”，而是：

- 存进去的顺序和取出来的顺序通常能对应起来

例如：

```java
List<String> list = new ArrayList<>();
list.add("A");
list.add("B");
list.add("C");
```

遍历时一般还是：

- `A`
- `B`
- `C`

## 3. 什么叫可重复

例如：

```java
list.add("A");
list.add("A");
```

这是允许的。

所以 `List` 不负责去重。

## 4. 常见实现类

当前阶段先记这 3 个：

- `ArrayList`
- `LinkedList`
- `Vector`

## 5. ArrayList

最常用。

特点可以先简单记成：

- 底层基于数组
- 查询快
- 尾部追加常用
- 日常开发最常见

## 6. LinkedList

特点先简单记成：

- 底层更偏链表结构
- 插入删除场景更有特点

当前阶段不用太深究底层细节，先记住它和 `ArrayList` 是不同实现就够了。

## 7. Vector

现在日常开发里使用较少。

常见认识：

- 比较老
- 线程安全

当前阶段只需知道它的存在，不用作为主力。

## 8. List 常用方法

### `add()`

添加元素。

```java
list.add("Tom");
```

### `get(index)`

按索引获取元素。

```java
String name = list.get(0);
```

### `set(index, element)`

替换指定位置元素。

### `remove()`

删除元素。

### `size()`

查看元素个数。

## 9. List 的典型使用场景

适合这些情况：

- 需要保留插入顺序
- 允许重复元素
- 需要按索引访问

例如：

- 评论列表
- 商品列表
- 学生名单

## 10. 当前阶段最该记住的结论

1. `List` 有序
2. `List` 可重复
3. 最常见实现类是 `ArrayList`
4. `ArrayList` 是日常开发高频集合

## 11. 一句话总结

如果你需要“按顺序存一批元素，并且允许重复”，第一反应通常就是 `List`。
