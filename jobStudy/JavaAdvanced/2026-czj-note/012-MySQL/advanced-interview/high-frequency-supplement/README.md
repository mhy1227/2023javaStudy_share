# MySQL 高频补漏专题

## 1. 为什么单独开这一组

这一组不是基础 MySQL 主线，也不是最常规的高频八股，而是那种：

- 社区面经里反复出现
- 一面二面都可能继续追
- 但很多人平时只零散听过，没有正式整理过

可以把这里理解成：

> **MySQL 面试里“主干都答完后，还容易被补刀”的那一层。**
