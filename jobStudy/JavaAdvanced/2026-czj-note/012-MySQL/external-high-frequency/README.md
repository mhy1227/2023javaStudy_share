# 外部面经高频题整理

## 1. 这一组文档是干什么的

这一组文档专门整理来自社区面经里的 MySQL 高频题。

和 `012-MySQL` 顶层文档的区别是：

- 顶层文档：更像系统复习主线。
- 这里的文档：更像从牛客、LeetCode、JavaGuide、SegmentFault 等来源里抽出来的“重复出现题型”。

## 2. 使用方式

建议这样配合：

1. 先看顶层主线文档，把知识点学顺。
2. 再看这里的高频题清单，确认哪些题在外部面经里反复出现。
3. 再看回答版，把答案调整成更适合面试口语输出的版本。
