# OpenFeign 与 LoadBalancer 高频问题

## 1. OpenFeign 是什么？

回答方向：

- 声明式 HTTP 调用组件
- 通过接口和注解描述远程调用
- 底层生成代理，把接口调用转换成 HTTP 请求
- 通常和注册中心、LoadBalancer 配合使用

## 2. 为什么要用 OpenFeign？

回答方向：

- 避免手写 HTTP 请求样板代码
- 提升服务调用代码可读性
- 统一服务间调用方式
- 方便整合负载均衡、超时、重试和降级

## 3. OpenFeign 和 RestTemplate 有什么区别？

回答方向：

- `RestTemplate` 偏手动 HTTP 调用
- `OpenFeign` 偏声明式接口调用
- Feign 更适合微服务内部调用
- RestTemplate 写法更底层，代码重复更多

## 4. OpenFeign 底层是怎么工作的？

回答方向：

- 扫描 `@FeignClient`
- 为接口生成动态代理
- 调用方法时解析注解和参数
- 根据服务名找到实例
- 结合负载均衡选择实例
- 发起 HTTP 请求并反序列化响应

## 5. Feign 调用是本地调用吗？

回答方向：

- 不是
- 只是写法上像本地接口调用
- 本质还是远程 HTTP 调用
- 需要考虑网络、超时、失败和降级

## 6. LoadBalancer 是什么？

回答方向：

- 客户端负载均衡组件
- 在多个服务实例中选择一个目标实例
- 通常结合注册中心使用
- 常见策略有轮询、随机、权重等

## 7. Nacos 和 LoadBalancer 是什么关系？

回答方向：

- Nacos 提供服务实例列表
- LoadBalancer 从实例列表里选择一个
- 两者不是替代关系
- 一句话：Nacos 告诉有哪些实例，LoadBalancer 决定打哪个实例

## 8. OpenFeign 和 LoadBalancer 是什么关系？

回答方向：

- Feign 负责声明式远程调用
- LoadBalancer 负责实例选择
- Feign 调用服务名时，LoadBalancer 选择具体 IP 和端口
- 两者合起来完成服务调用链路

## 9. 客户端负载均衡和服务端负载均衡有什么区别？

回答方向：

- 客户端负载均衡：调用方本地选择实例
- 服务端负载均衡：请求先到代理层，再由代理层转发
- Spring Cloud LoadBalancer 属于客户端负载均衡
- Nginx、SLB、网关更偏服务端负载均衡

## 10. Feign 超时怎么理解？

回答方向：

- 远程调用必须设置超时
- 避免调用方线程长时间阻塞
- 常见有连接超时和读取超时
- 超时不是为了保证成功，而是为了控制失败成本

## 11. Feign 重试能不能随便开？

回答方向：

- 不能
- 重试可能放大下游压力
- 写操作和非幂等接口不能随便重试
- 更适合短暂网络抖动和幂等读请求

## 12. Feign 调用失败怎么办？

回答方向：

- 配置超时
- 谨慎重试
- 异常统一处理
- 结合 Sentinel 做熔断降级
- 对非核心功能返回兜底结果

## 13. 如果某个服务实例挂了，LoadBalancer 会马上不选它吗？

回答方向：

- 不一定马上
- 依赖注册中心健康检查和实例列表刷新
- 中间存在时间差
- 所以仍然需要超时、重试、熔断等兜底

## 14. 如果注册中心没有目标服务实例会怎样？

回答方向：

- Feign 调用会失败
- 常见原因是服务没启动、注册失败、服务名写错、网络问题
- 排查方向是注册中心实例列表、服务名、健康检查和网络

## 15. OpenFeign 能不能传递请求头？

回答方向：

- 可以
- 常见场景是用户身份、链路追踪 ID、灰度标识
- 可以通过拦截器统一透传
- 要注意不要透传敏感或无关头信息

## 16. Feign 日志有什么用？

回答方向：

- 用于排查请求路径、参数、响应和异常
- 开发和测试环境可以适当打开详细日志
- 生产环境要注意日志量和敏感信息

## 17. OpenFeign 和 Dubbo 有什么区别？

回答方向：

- OpenFeign 通常是 HTTP REST 调用
- Dubbo 更偏 RPC 框架
- Dubbo 在高性能 RPC 和服务治理上更完整
- Spring Cloud 体系里 Feign 使用更自然

## 18. 项目里你会怎么介绍这块？

回答方向：

- 服务之间通过 OpenFeign 调用
- Nacos 做注册发现
- LoadBalancer 做客户端负载均衡
- 配置了超时和异常处理
- 核心链路结合 Sentinel 做熔断降级
