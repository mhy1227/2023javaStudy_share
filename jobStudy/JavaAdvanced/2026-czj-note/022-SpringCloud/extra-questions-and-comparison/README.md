# Spring Cloud 扩展问题与组件对比

## 1. 这个目录放什么

这个目录专门放 Spring Cloud 主体之外的扩展问题。

主要包括：

- 牛客、LeetCode、实习僧等面经里可能出现的补充问题
- 组件横向对比
- 版本演进问题
- 链路追踪与监控
- 服务拆分实践题

## 2. 为什么单独开目录

因为这些问题不完全属于某一个组件。

比如：

- Spring Cloud 和 Spring Boot 区别
- Spring Cloud 和 Dubbo 区别
- Netflix 和 Alibaba 体系区别
- Hystrix 为什么逐渐被 Sentinel 替代
- Gateway 和 Zuul 差异
- 微服务怎么排查慢调用

这些更适合放在横向扩展目录里。
