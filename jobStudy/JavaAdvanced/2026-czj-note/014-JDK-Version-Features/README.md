# 014-JDK-Version-Features

## 1. 这一章是干什么的

这一章专门整理：

- `JDK 8`
- `JDK 9`
- `JDK 17`

这几个版本里，面试最常问、最值得记的特性。

它和 `013-JVM` 不一样。

- `013-JVM` 更偏运行时、内存、类加载、GC
- 这一章更偏语言和平台能力的版本演进

## 2. 为什么值得单独开目录

因为这类题在面试里很常见：

- JDK 8 有哪些新特性
- JDK 9 你知道什么
- JDK 17 为什么重要
- 你项目里为什么还停在 8 / 为什么升级到 17
- 版本升级时有哪些明显变化

如果这些内容散在 JVM 或 Java 基础里，复习会很乱，所以单独开目录更合适。
