# Spring Boot 20 个核心问题

## 1. 这份文档怎么用

这一篇把 Spring Boot 压缩成 20 个最核心的问题。

适合：

- 一面前快速抽问
- 最后一轮冲刺复习
- 判断自己到底是卡在“定位题”还是“自动装配题”

## 2. 核心 20 题

1. 什么是 Spring Boot？
2. Spring Boot 和 Spring 是什么关系？
3. 为什么有了 Spring 还要 Spring Boot？
4. Spring、Spring MVC、Spring Boot 三者关系怎么讲？
5. Spring Boot 最核心解决了什么问题？
6. 什么叫约定优于配置？
7. 什么是自动装配？
8. 自动装配为什么能减少配置？
9. 为什么自动装配不是“全都自动配”？
10. 什么叫按条件装配？
11. `@SpringBootApplication` 是什么？
12. `@SpringBootApplication` 里大概包含什么能力？
13. `@EnableAutoConfiguration` 是干什么的？
14. 常见条件注解有哪些？
15. `starter` 是什么？
16. `starter` 和自动装配是什么关系？
17. Spring Boot 启动流程大概怎么讲？
18. Spring Boot 3 在自动配置文件上有什么变化？
19. 如果结合项目，你怎么解释 Spring Boot 的价值？
20. 如果没有 Spring Boot，项目哪里会更麻烦？
