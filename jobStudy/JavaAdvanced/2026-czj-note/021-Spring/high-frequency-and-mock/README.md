# Spring 高频问答与模拟目录

## 1. 这个子目录是干什么的

`021-Spring` 顶层现在主要放正式主线文档：

- Spring 是什么
- IOC / AOP / Bean 生命周期
- 事务 / 事务失效
- MVC
- 常见注解
- 循环依赖
- Spring Boot 基础
- 自动装配

这些文档适合系统复习。

但真正面试时，还需要另一套内容：

- 高频追问题
- 口语化回答
- 连续追问链
- mock 脚本

所以这里单独开一个子目录，把“主线知识”和“面试回答”分开。

## 2. 当前这层怎么用

建议这样用：

1. 先把顶层主线文档看完
2. 再回到这个目录刷高频题
3. 再看口语版回答，练“怎么说”
4. 最后再补 mock 和连续追问
