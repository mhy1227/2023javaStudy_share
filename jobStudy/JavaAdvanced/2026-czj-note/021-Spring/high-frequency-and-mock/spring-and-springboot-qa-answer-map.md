# Spring / Spring Boot 问题索引图

## 1. 这份文档怎么用

这一篇不是新知识，而是复习索引。

目标只有一个：

当你脑子里只剩一个问题时，能立刻知道：

- 这个问题该看哪篇
- 它对应哪类口语回答
- 它属于哪条主线

也就是把：

> 问题 -> 文档 -> 口语回答

这条映射关系做出来。
