# Spring 对比专题

## 1. 这个子目录是干什么的

`021-Spring` 顶层现在已经有：

- 主线文档
- 高频问答与 mock

但还有一类内容，复习时非常高频：

- 各种概念对比
- 常见注解对比
- Spring / MVC / Boot 之间的关系对比
- 启动、事件、Runner 这类容易混的点

所以这里单独开一个“对比专题”子目录，把这些内容统一收口。

## 2. 这一层怎么用

建议用法：

1. 先看顶层主线文档
2. 再刷高频问答
3. 最后回到这个目录，专门做“易混概念纠偏”

## 3. 当前已经放了什么

- `spring-compare-note.md`
- `spring-compare-long-answer-note.md`
- `spring-compare-oral-pack.md`

其中：

- `spring-compare-note.md` 更偏速记版
- `spring-compare-long-answer-note.md` 更偏面试展开版
- `spring-compare-oral-pack.md` 更偏一面口语版

## 4. 一句话记住

> 这一层不是讲新知识，而是把已经学过的知识做“并排对比”，防止面试时说混。
